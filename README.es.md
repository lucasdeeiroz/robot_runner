# 🤖 Robot Runner

**El Centro de Comando Profesional para Automatización en Android.**

Robot Runner es un ecosistema de escritorio de alto rendimiento diseñado para acortar la brecha entre la exploración manual y la automatización profesional.

## 🌟 La Ventaja QA: ¿Por qué Robot Runner?

En un flujo tradicional, un ingeniero de QA cambia constantemente de ventanas. **Robot Runner elimina este cambio de contexto.** Proporciona un entorno integrado donde los diagnósticos y el diseño de la automatización ocurren en paralelo.

---

## 🚀 Capacidades Principales

### 🎛️ Centro de Comando Unificado
*   **Descubrimiento Inteligente**: Detección instantánea de dispositivos USB y Wi-Fi.
*   **Libertad Inalámbrica**: Emparejamiento perfecto mediante TCP/IP.
*   **Alcance Global**: Túnel **ngrok** integrado para probar dispositivos remotos.
*   **Duplicación de Alta Fidelidad**: Duplicación de pantalla de latencia ultrabaja (Scrcpy).

### 🔍 Inspección de UI de Precisión
*   **Generación de Localizadores**: Genere al instante XPaths y métodos `UiSelector` optimizados.
*   **Selectores Inteligentes de IA**: Evalúe y extraiga automáticamente el localizador más resistente.

### 📊 Diagnósticos en Tiempo Real
*   **Logcat Inteligente**: Registros del sistema y aplicaciones con filtrado avanzado.
*   **Información de Rendimiento**: Rastree el uso de CPU, RAM y batería en tiempo real.

### 🧠 Diseño y Mapeo Guiados por IA
*   **Mapeo Autónomo**: Use el Mapeador para crear un gemelo digital de su aplicación.
*   **IA de Prompt a Código**: Convierta el lenguaje natural en scripts sólidos de BDD (Robot Framework).

### ⚡ Ejecución de Nivel Profesional
*   **Modos de Ejecución**: Soporte para archivos `.robot`, directorios y `.args`.
*   **Consola en Vivo**: Vea la ejecución estándar directamente dentro de la aplicación.
*   **Perfiles de Configuración**: Cambie al instante entre proyectos y claves de API.
