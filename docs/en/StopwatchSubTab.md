# Toolbox: Stopwatch & Benchmarks

A precision timing utility for manual profiling and benchmarking app transitions.

### Key Features

- **Lap Timers:** Track how long specific transitions (like Login times, cold start times) take.
- **Delta Comparisons:** Compare lap times to measure regressions in UI responsiveness.

### How to Use
Used primarily during manual exploratory testing alongside the Inspector, when investigating reported slowness in the app.
