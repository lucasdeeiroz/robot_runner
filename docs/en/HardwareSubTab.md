# Toolbox: Hardware Stats

The Hardware tab displays a deep dive into the physical status of the device.

### Key Features

- **Battery Diagnostics:** View current voltage, temperature, technology type, and remaining charge level.
- **CPU Architecture:** Check core counts, architecture (arm64, x86), and current frequencies.
- **Memory Breakdown:** Visualize total RAM versus available RAM, critical for understanding Out-Of-Memory (OOM) app crashes.

### How to Use
Navigate to this tab when diagnosing performance bottlenecks or when an app behaves unexpectedly on a specific manufacturer's device.
