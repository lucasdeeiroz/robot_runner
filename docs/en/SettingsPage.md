# Settings Page

The global configuration hub for Robot Runner, managing everything from binary paths to AI credentials.

### Key Features

- **Profile Management:** Create isolated configurations (e.g., 'Local Environment', 'CI Environment') and switch between them instantly. Import/Export setups to share with your team.
- **Path Configurations:** Define where your `Automation Root`, `Appium Binary`, `Log Output`, and `Screenshots` are stored.
- **Integrations:** Configure connections to external tools like Jira, Azure DevOps, and TestLink for automated reporting.
- **AI Providers:** Inject API keys for Gemini, Claude, or OpenAI. Select preferred models and test integrations.
- **Appearance & Network:** Toggle Dark/Light mode, change Appium Host/Port, and configure global UI behaviors.

### How to Use
Accessible via the gear icon in the sidebar. Remember that any change to Paths requires you to verify them; the UI provides check-icons to validate if the provided paths exist.
