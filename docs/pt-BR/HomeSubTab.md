# Início (Dashboard)

A aba Início é a central de comando do Robot Runner. Ela fornece uma visão geral imediata do seu ambiente de teste, status dos dispositivos e acesso rápido às funcionalidades principais.

### Principais Funcionalidades

- **Descoberta de Dispositivos:** Detecta e exibe automaticamente todos os dispositivos USB e Wi-Fi conectados. Cada card de dispositivo mostra seu modelo, versão do SO e status de conexão.
- **Monitoramento de Servidores:** Em segundo plano, verifica continuamente se os servidores ADB e Appium estão rodando.
- **Ações Rápidas:** 
  - **Tirar Screenshot:** Capture instantaneamente a tela de um dispositivo selecionado.
  - **Reiniciar Servidores:** Mate e reinicie rapidamente o ADB ou Appium se eles travarem.
- **Histórico de Execução:** Um gráfico rápido mostrando a proporção PASS/FAIL das suas execuções mais recentes.

### Como Usar
1. Certifique-se de que seu dispositivo está conectado via USB ou Wi-Fi.
2. Aguarde alguns segundos para o dashboard atualizar e exibir o card do dispositivo.
3. Clique no card do dispositivo para selecioná-lo como o alvo ativo para as próximas ações.
