# Histórico

A aba Histórico fornece uma trilha de auditoria abrangente de suas execuções de teste anteriores.

### Principais Funcionalidades

- **Log de Execução:** Uma lista paginada de rodadas passadas, exibindo horário, dispositivo alvo, modo e status (PASS/FAIL).
- **Visualização de Métricas:** Gráficos que oferecem insights sobre suas taxas de sucesso ao longo do tempo.
- **Leitura de Logs:** Leia diretamente o conteúdo dos arquivos `output.xml` e `log.html` gerados.
- **Re-execução em um clique:** Repita facilmente uma execução passada com os mesmos parâmetros exatos.

### Como Usar
1. Selecione uma linha de execução para expandir seus detalhes.
2. Use o botão 'Ver Log' para abrir o relatório HTML.
3. Use os filtros no topo para buscar testes por status ou data.
