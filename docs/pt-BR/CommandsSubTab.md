# Toolbox: Comandos ADB

A aba Comandos atua como um terminal integrado especializado em enviar instruções ADB diretas para o dispositivo.

### Principais Funcionalidades

- **Execução Shell Direta:** Envie comandos `adb shell` puros diretamente ao aparelho, sem abrir terminais externos.
- **Snippets Salvos:** Salve comandos frequentes (ex: abrir deep links, disparar notificações) para uso rápido.
- **Parsing de Saída:** Exibe os logs de retorno stdout/stderr de forma limpa na interface.

### Como Usar
Digite seu comando (ex: `am start -a android.intent.action.VIEW -d 'myapp://home'`) e clique em 'Executar'. O resultado será exibido instantaneamente.
