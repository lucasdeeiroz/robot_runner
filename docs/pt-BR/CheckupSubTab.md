# Toolbox: Checkup do Dispositivo

A aba Checkup avalia a saúde geral e a prontidão do dispositivo para automação de testes.

### Principais Funcionalidades

- **Saúde da Rede:** Faz ping em servidores externos para garantir que o dispositivo tenha internet ativa.
- **Verificação de Armazenamento:** Alerta se o armazenamento interno estiver criticamente baixo, o que poderia causar travamentos nos testes.
- **Pré-requisitos de Automação:** Valida se as Opções de Desenvolvedor e os componentes do Appium estão instalados corretamente no aparelho.

### Como Usar
Abra a aba e clique em 'Executar Diagnóstico'. O sistema gerará um relatório destacando problemas críticos que devem ser corrigidos antes de testar.
