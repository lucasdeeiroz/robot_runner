# Console de Execução

O Console de Execução é um painel que exibe a saída padrão (stdout/stderr) de suas suítes de teste em tempo real.

### Principais Funcionalidades

- **Espelhamento de Terminal:** Veja a saída exatamente como o Robot Framework ou Appium a produz.
- **Cores ANSI e Auto-Scroll:** Suporta códigos de cor de terminal padrão para leitura fácil de tags PASS/FAIL.
- **Abortar Execução:** Apresenta um botão de pânico para matar forçadamente uma rodada de teste que travou.

### Como Usar
Sempre que uma execução de teste é disparada na aba Testes, o Console de Execução é ativado automaticamente para transmitir o contexto.
