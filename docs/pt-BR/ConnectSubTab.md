# Conectar

A aba Conectar é especializada em conexões remotas e sem fio de dispositivos, incluindo recursos avançados de tunelamento.

### Principais Funcionalidades

- **ADB sem Fio:** Conecte-se a dispositivos via Wi-Fi sem precisar de cabo USB. Insira o IP e a porta (padrão 5555).
- **Pareamento Android 11+:** Suporte a códigos de pareamento para conexões sem fio inéditas.
- **Túnel Ngrok:** Exponha seu servidor Appium local de forma segura à internet, vital para testar dispositivos em redes diferentes.
- **Descoberta Automática:** Obtém automaticamente o endereço IP ativo do dispositivo se ele já estiver conectado via USB.

### Como Usar
1. **ADB Wi-Fi:** Com o dispositivo no USB, o IP será preenchido. Clique em 'Conectar' para estabelecer a conexão TCP/IP, e então remova o cabo.
2. **Túnel Ngrok:** Clique em 'Iniciar Ngrok'. Uma URL segura será gerada para uso remoto.
