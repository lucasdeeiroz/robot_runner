# Página de Configurações

O hub de configuração global do Robot Runner, gerenciando desde caminhos de binários até credenciais de IA.

### Principais Funcionalidades

- **Gerenciamento de Perfis:** Crie configurações isoladas (ex: 'Ambiente Local', 'Ambiente CI') e alterne instantaneamente. Importe/Exporte para compartilhar com a equipe.
- **Configurações de Caminhos:** Defina onde ficam seu `Automation Root`, `Binário do Appium`, e `Screenshots`.
- **Integrações:** Configure conexões para ferramentas externas como Jira e Azure DevOps.
- **Provedores de IA:** Insira chaves de API para Gemini, Claude ou OpenAI. Selecione modelos.
- **Aparência e Rede:** Alterne Dark Mode e configure Host/Port do Appium.

### Como Usar
Acessível via o ícone de engrenagem na barra lateral. Qualquer mudança nos Caminhos é validada em tempo real com ícones de verificação.
