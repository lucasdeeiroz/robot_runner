# Toolbox: Dmesg (Logs de Kernel)

A aba Dmesg fornece acesso aos logs de baixo nível do kernel do Android.

### Principais Funcionalidades

- **Depuração de Hardware/Drivers:** Extremamente útil ao testar aplicativos IoT ou integrações Bluetooth/BLE complexas.
- **Stream em Tempo Real:** Escuta continuamente o buffer circular do kernel.
- **Exportação:** Exporte os logs brutos para análise externa.

### Como Usar
Abra a aba para iniciar o stream automático dos logs de kernel. Use a barra de pesquisa para filtrar tags específicas de drivers.
