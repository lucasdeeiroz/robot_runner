# Toolbox: Status de Hardware

A aba Hardware exibe um mergulho profundo no status físico do dispositivo.

### Principais Funcionalidades

- **Diagnóstico de Bateria:** Veja a voltagem atual, temperatura, tecnologia e carga restante.
- **Arquitetura de CPU:** Verifique a quantidade de núcleos, arquitetura (arm64, x86) e frequências.
- **Consumo de Memória:** Visualize a RAM total versus RAM disponível, crucial para investigar crashes de Out-Of-Memory (OOM).

### Como Usar
Navegue até esta aba para diagnosticar gargalos de performance ou quando o app apresentar comportamento bizarro em dispositivos específicos.
