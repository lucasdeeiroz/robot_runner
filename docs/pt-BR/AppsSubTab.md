# Toolbox: Gerenciador de Apps

A aba Apps dentro da Toolbox permite controle granular sobre os pacotes instalados no dispositivo conectado.

### Principais Funcionalidades

- **Listar Pacotes:** Visualize uma lista categorizada de apps de usuário e de sistema.
- **Instalação de APK:** Instale arquivos `.apk` locais no dispositivo, com suporte a forçar downgrade e conceder permissões automaticamente.
- **Ações de Ciclo de Vida:**
  - **Limpar Dados:** Limpa cache e dados do app, resetando-o (útil para cenários de QA limpos).
  - **Desinstalar/Desabilitar:** Remova ou congele pacotes.
  - **Iniciar:** Força o início da Main Activity do aplicativo.

### Como Usar
Use a barra de pesquisa para encontrar seu pacote alvo. Clique nos botões de ação para limpar dados ou desinstalar antes de iniciar uma nova rodada de testes.
