# Gerador de IA

O Gerador de IA utiliza Grandes Modelos de Linguagem para acelerar a criação de seus scripts de teste.

### Principais Funcionalidades

- **Prompt para Código:** Escreva uma descrição (ex: 'Crie um teste de login válido') e a IA gerará a sintaxe Robot Framework BDD.
- **Conversão de Gravações:** Importe macros gravadas no Inspetor. A IA converterá os cliques em Keywords estruturadas usando melhores práticas de Page Object Model.
- **Injeção de Contexto:** A IA tem consciência das suas Configurações, adaptando o idioma e as restrições do projeto na resposta gerada.

### Como Usar
1. Configure uma chave de API nas Configurações.
2. Escreva seu prompt ou importe uma sessão gravada.
3. Clique em 'Gerar' e revise o código `.robot` gerado.
