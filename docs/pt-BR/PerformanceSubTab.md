# Toolbox: Monitor de Performance

Gráficos em tempo real do consumo de recursos específicos do aplicativo durante a execução do teste.

### Principais Funcionalidades

- **Gráficos ao Vivo:** Observe o uso de CPU % e RAM (MB) plotados em uma linha do tempo.
- **Monitoramento Específico:** Ao contrário de monitores globais, ele rastreia especificamente o consumo do aplicativo selecionado.
- **Exportação CSV:** Grave uma sessão de performance enquanto roda seus testes e exporte os dados para relatórios de baseline.

### Como Usar
Selecione seu pacote, clique em 'Iniciar Gravação' e execute sua suíte de testes. Analise picos e vales para encontrar vazamentos de memória.
