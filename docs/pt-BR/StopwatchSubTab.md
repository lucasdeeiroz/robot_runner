# Toolbox: Cronômetro

Um utilitário de cronometragem de precisão para profiling manual e benchmarking de transições do aplicativo.

### Principais Funcionalidades

- **Tempos de Volta (Laps):** Rastreie quanto tempo transições específicas (como o Login, ou Cold Start) demoram.
- **Comparações Delta:** Compare os tempos para medir regressões na responsividade da UI.

### Como Usar
Utilizado principalmente durante testes exploratórios manuais junto com o Inspetor, ao investigar lentidão reportada no app.
