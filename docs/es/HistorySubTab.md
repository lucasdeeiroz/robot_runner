# Historial

La pestaña Historial proporciona un registro de auditoría completo de sus ejecuciones de pruebas anteriores.

### Características Clave

- **Registro de Ejecución:** Una lista de ejecuciones pasadas, que muestra la marca de tiempo, el dispositivo de destino y el estado (PASS/FAIL).
- **Visualización de Métricas:** Gráficos que ofrecen información sobre sus tasas de aprobación a lo largo del tiempo.
- **Análisis de Registros:** Lea directamente el contenido de los archivos `output.xml` y `log.html` generados.
- **Re-ejecutar con un clic:** Vuelva a activar fácilmente una ejecución pasada con los mismos parámetros.

### Cómo Usar
1. Seleccione una fila de ejecución para expandir sus detalles.
2. Use el botón 'Ver Registro' para abrir el informe HTML.
3. Use los filtros en la parte superior para buscar resultados por estado o fecha.
