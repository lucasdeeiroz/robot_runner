# Conectar

La pestaña Conectar se especializa en conexiones de dispositivos remotos e inalámbricos, incluidas funciones avanzadas de túneles.

### Características Clave

- **ADB Inalámbrico:** Conéctese a dispositivos a través de Wi-Fi sin necesidad de un cable USB.
- **Emparejamiento Android 11+:** Soporte para códigos de emparejamiento para nuevas conexiones inalámbricas.
- **Túnel Ngrok:** Exponga su servidor Appium local de forma segura a Internet.
- **Descubrimiento Automático:** Obtiene automáticamente la dirección IP activa de un dispositivo si ya está conectado a través de USB.

### Cómo Usar
1. **ADB Wi-Fi:** Si su dispositivo está conectado a través de USB, su IP se completará automáticamente. Haga clic en 'Conectar' para establecer la conexión.
2. **Túnel Ngrok:** Haga clic en 'Iniciar Ngrok' para generar una URL segura.
