# Toolbox: Logcat

Una implementación de alto rendimiento del Logcat de Android, diseñada para ingenieros de control de calidad.

### Características Clave

- **Virtualización:** Capaz de procesar decenas de miles de líneas de registro sin ralentizar la interfaz de usuario.
- **Filtrado de Paquetes:** Aísle automáticamente los registros que pertenecen exclusivamente a su aplicación, filtrando el ruido del sistema.
- **Niveles de Registro y Regex:** Filtre visualmente por Nivel de Registro y use Expresiones Regulares para coincidencias potentes.
- **Pausar/Reanudar:** Congele la transmisión del registro para investigar una excepción.

### Cómo Usar
1. Seleccione su paquete de aplicación objetivo en el menú desplegable.
2. La vista filtrará automáticamente el caos de los registros de Android.
