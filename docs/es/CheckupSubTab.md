# Toolbox: Chequeo del Dispositivo

La pestaña Checkup evalúa la salud general y la preparación del dispositivo para la automatización de pruebas.

### Características Clave

- **Salud de la Red:** Hace ping a servidores externos para garantizar que el dispositivo tenga acceso a Internet.
- **Comprobaciones de Almacenamiento:** Advierte si el almacenamiento interno es peligrosamente bajo.
- **Requisitos de Automatización:** Valida si las Opciones de Desarrollador y los componentes del servidor Appium están instalados correctamente.

### Cómo Usar
Abra la pestaña y haga clic en 'Ejecutar Diagnóstico'. El sistema generará un informe que destaca cualquier problema crítico que deba abordarse.
