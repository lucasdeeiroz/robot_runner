# Toolbox: Cronómetro

Una utilidad de cronometraje de precisión para la creación manual de perfiles y la evaluación comparativa de las transiciones de la aplicación.

### Características Clave

- **Temporizadores de Vuelta:** Realice un seguimiento del tiempo que tardan las transiciones específicas (como tiempos de inicio de sesión).
- **Comparaciones Delta:** Compare los tiempos de vuelta para medir las regresiones en la capacidad de respuesta de la interfaz de usuario.

### Cómo Usar
Se utiliza principalmente durante pruebas exploratorias manuales junto con el Inspector.
