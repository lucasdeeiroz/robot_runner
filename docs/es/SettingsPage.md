# Página de Configuración

El centro de configuración global para Robot Runner, que administra todo, desde rutas de binarios hasta credenciales de IA.

### Características Clave

- **Administración de Perfiles:** Cree configuraciones aisladas (por ejemplo, 'Entorno Local') y cambie entre ellas al instante. Importe/Exporte configuraciones.
- **Configuraciones de Rutas:** Defina dónde se almacenan su Directorio Raíz y Capturas de Pantalla.
- **Integraciones:** Configure conexiones a herramientas externas como Jira y Azure DevOps.
- **Proveedores de IA:** Inyecte claves API para Gemini, Claude u OpenAI.
- **Apariencia:** Cambie entre el modo Oscuro/Claro y configure Host/Puerto de Appium.

### Cómo Usar
Accesible a través del icono de engranaje en la barra lateral. Cualquier cambio en las Rutas debe verificarse.
