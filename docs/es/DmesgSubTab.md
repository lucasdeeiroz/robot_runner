# Toolbox: Dmesg (Registros del Kernel)

La pestaña Dmesg proporciona acceso a los registros del kernel de Android de bajo nivel.

### Características Clave

- **Depuración de Hardware y Controladores:** Extremadamente útil cuando se prueban aplicaciones IoT complejas o integraciones Bluetooth.
- **Transmisión en Tiempo Real:** Sondea continuamente el búfer de anillo del kernel.
- **Exportar:** Exporte registros sin formato para depuración avanzada.

### Cómo Usar
Abra la pestaña para transmitir automáticamente los registros del kernel. Use la barra de búsqueda para filtrar etiquetas específicas de controladores.
