# Consola de Ejecución

La Consola de Ejecución es un panel que muestra la salida estándar de sus suites de pruebas en ejecución en tiempo real.

### Características Clave

- **Duplicación de Terminal:** Vea la salida exactamente como la produce Robot Framework.
- **Colores ANSI y Desplazamiento Automático:** Admite códigos de color de terminal estándar para facilitar la lectura.
- **Abortar Ejecución:** Cuenta con un botón de pánico para detener por la fuerza una ejecución de prueba bloqueada.

### Cómo Usar
Cada vez que se activa una ejecución de prueba, la Consola de Ejecución se activa automáticamente para transmitir el contexto.
