# Toolbox: Comandos ADB

La pestaña Comandos actúa como un terminal integrado especializado en enviar instrucciones ADB directas al dispositivo.

### Características Clave

- **Ejecución Directa de Shell:** Envíe comandos `adb shell` en bruto directamente al dispositivo.
- **Fragmentos Guardados (Snippets):** Guarde comandos utilizados con frecuencia para una ejecución rápida.
- **Análisis de Salida:** Muestra los registros de retorno de forma limpia dentro de la interfaz de usuario.

### Cómo Usar
Escriba su comando (por ejemplo, `am start -a android.intent.action.VIEW -d 'myapp://home'`) y presione 'Ejecutar'.
